How to run Cryptocurrency Buzz program?
Please go to folder /group16/cryptoAnalyzer/src/main/java/cryptoAnalyzer/mainEntrance 
and run CryptocurrencyBuzzEntrance.java

I encounter a problem says"Launching CryptocurrencyBuzzEntrance has encountered a problem.
To solve this problem,
1. Right click CryptocurrencyBuzzEntrance.java
2. Select build path - configure build path - java build path - libraries
3. Remove json-simple-1.1.jar
4. Apply and close
5. Run CryptocurrencyBuzzEntrance.java
